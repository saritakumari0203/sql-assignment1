# College Academic Database Management System (DBMS Lab)

## Institution

B.P. Mandal College of Engineering

---

## Project Description

This project demonstrates the implementation of a **College Academic Database System** using **Oracle SQL**.
The system manages academic data including **Departments, Faculty, Courses, Students, and Enrollment records** with proper **Primary Key and Foreign Key constraints** to maintain referential integrity.

---

## Objectives

* Design relational tables from ER model
* Implement database schema using SQL
* Insert sample records into each table
* Maintain relationships between tables using constraints
* Retrieve and display academic information using SQL queries

---

## Database Tables

The following tables are implemented in this project:

1. **Department** – Stores department details
2. **Faculty** – Stores faculty information linked to departments
3. **Course** – Contains course details with assigned faculty
4. **Student** – Stores student personal and academic data
5. **Enrollment** – Maintains student course registration and grades

---

## Sample Operations Performed

* Table creation using `CREATE TABLE`
* Data insertion using `INSERT INTO`
* Table structure verification using `DESC`
* Data retrieval using `SELECT`
* Output formatting using `SET LINESIZE`, `COLUMN FORMAT`
